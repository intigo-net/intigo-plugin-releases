## Intigo WordPress Plugin v0.1.0

Initial public release of the Intigo WooCommerce plugin package.

### Included in this release

- `intigo-wordpress-plugin-v0.1.0.zip`
- `SHA256SUMS.txt`

### Functional highlights

- Parcel creation from WooCommerce orders
- Bordereau generation, download, and print actions
- Metadata edit workflow (city, district, neighborhood, package)
- Sync and operational parcel actions from order management UI
- Dashboard, logs, documentation, and settings interfaces
- Sandbox/Production mode handling with fixed Intigo API endpoints
- In-app status/feedback notifications for key actions

### Installation

1. Download `intigo-wordpress-plugin-v0.1.0.zip` from release assets.
2. In WordPress admin: **Plugins -> Add New -> Upload Plugin**.
3. Upload the zip, install, activate.
4. Open Intigo settings and configure API key + environment mode.

### Compatibility

- WordPress: modern 6.x versions
- WooCommerce: modern 8.x/9.x versions
- PHP: 8.x recommended
