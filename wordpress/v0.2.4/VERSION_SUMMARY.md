# WordPress v0.2.4 - Version Summary

## What changed

### Version badge on the admin interface

- The installed plugin version (e.g. `v0.2.4`) now shows as a pill in the header of **every Intigo admin page** — Dashboard, Manage Orders, Logs, Documentation, Settings — next to the API and environment status pills.
- Driven by the `INTIGO_PARCELS_VERSION` constant, so it always reflects the running version. Useful for confirming at a glance which version a store has installed.

_No changes to checkout, pricing, order push, or the size resolver._

## Artifact

- `wordpress-intigo-parcels-v0.2.4.zip`
