# WordPress v0.1.0 - Version Summary

## What changed in this version

- Initial public packaging of the Intigo WordPress/WooCommerce plugin.
- Added order-level actions for parcel lifecycle:
  - create parcel
  - synchronize updates
  - generate bordereau
  - report delay / relance / cancel (via actions flow)
- Added operational interfaces in WordPress admin:
  - Dashboard
  - Manage Orders
  - Logs
  - Documentation
  - Settings
- Added sandbox/production mode control and API-key based configuration.
- Added status feedback UX and improved metadata editing flow.
